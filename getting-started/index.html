This is my personal API platform, that I use to manage API Evangelist, and all the other projects I'm working on. This will always be my master stack, with smaller projects broken down into separate, more meaningful portals.

Here are the steps to get started:

  1. Register for a developer account
  2. Visit the documentation for the API you wish to integrate with
  3. Visit the code page, and download the language library you need
  4. Import postman collections into your postman if you need to play further
  5. Submit a Github issue for the master repository, or each specific API if you need help

I am continuing to work on the APis, and improve the design, error codes, and underlying data models. I am also working in real-time on the supporting building blocks like documentation, roadmap, etc.

If there is anything I can make smoother, or left out please let me know.
